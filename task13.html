# Even-Odd Number Checker ## 📌 Description A simple JavaScript program that checks whether a given number is **even** or **odd** using the modulus operator (`%`). It demonstrates the use of variables, conditional statements (`if-else`), and `console.log()` for output. ## 🚀 Features - Input a number and determine if it is even or odd. - Beginner-friendly example of JavaScript conditionals. - Clean and minimal code structure. ## 🛠️ Code Example ```javascript const number = num; if (num % 2 === 0) { console.log("Number is even"); } else { console.log("Number is odd"); }
